# Employee Performance Management System (Oracle PL/SQL)

## Project Overview

This project is an Employee Performance Management System built using Oracle PL/SQL and the pre-existing Oracle HR schema.

It extends the HR schema by introducing:

* Employee performance tracking using letter grades (A, B, C, D, F)
* Performance-based bonus calculation using PL/SQL logic
* Automated performance report generation
* Clean business-rule implementation using conditional statements

The project demonstrates:

* Table design with constraints
* Foreign key relationships
* Cursor FOR loops
* Conditional logic (IF / ELSIF / ELSE)
* SQL joins
* Formatted reporting output
* Real-world HR business logic simulation

Developed and tested using Oracle Database and TOAD for Oracle.

# Database Design

## 1. Performance Reviews Table

```sql
CREATE TABLE hr.performance_reviews (
    review_id         NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    employee_id       NUMBER REFERENCES hr.employees(employee_id),
    review_period     DATE,
    performance_grade CHAR(1) CHECK (performance_grade IN ('A','B','C','D','F')),
    comments          VARCHAR2(200)
);
```

---

## Table Breakdown

### review_id

* Auto-generated primary key using IDENTITY
* Ensures each review is unique

### employee_id

* Foreign key referencing hr.employees
* Enforces referential integrity
* Guarantees all reviews belong to valid employees

### review_period

* Represents the evaluation date
* Can be monthly, quarterly, or yearly

### performance_grade

* Stores letter grade only
* Restricted using CHECK constraint

Meaning of grades:

* A = Excellent
* B = Very Good
* C = Average
* D = Below Average
* F = Fail

This prevents invalid grades from being inserted.

### comments

* Optional manager evaluation notes

# Core Business Logic

## Main Procedure: generate_performance_report

```plsql
CREATE OR REPLACE PROCEDURE generate_performance_report(
    p_period IN DATE
) IS

    v_bonus      NUMBER;
    v_bonus_pct  NUMBER;

BEGIN
    DBMS_OUTPUT.PUT_LINE('Employee Performance Report for ' || TO_CHAR(p_period,'YYYY-MM'));
    DBMS_OUTPUT.PUT_LINE('--------------------------------------------------------------------------------');

    FOR emp IN (
        SELECT e.employee_id,
               e.first_name,
               e.last_name,
               e.salary,
               pr.performance_grade
        FROM hr.employees e
        JOIN hr.performance_reviews pr
          ON e.employee_id = pr.employee_id
        WHERE pr.review_period = p_period
    )
    LOOP

        IF emp.performance_grade = 'A' THEN
            v_bonus_pct := 0.20;

        ELSIF emp.performance_grade = 'B' THEN
            v_bonus_pct := 0.10;

        ELSIF emp.performance_grade = 'C' THEN
            v_bonus_pct := 0.05;

        ELSIF emp.performance_grade = 'D' THEN
            v_bonus_pct := 0.02;

        ELSE
            v_bonus_pct := 0;
        END IF;

        v_bonus := emp.salary * v_bonus_pct;

        DBMS_OUTPUT.PUT_LINE(
            emp.employee_id || ' | ' ||
            emp.first_name || ' ' || emp.last_name ||
            ' | Salary: ' || TO_CHAR(emp.salary, '999,999') ||
            ' | Grade: ' || emp.performance_grade ||
            ' | Bonus %: ' || (v_bonus_pct * 100) || '%' ||
            ' | Final Bonus: ' || TO_CHAR(v_bonus, '999,999')
        );

    END LOOP;

END;
/
```
